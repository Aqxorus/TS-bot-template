# TS template

First, put your bot token in the .env file:

```sh
# .env file example
TOKEN=ABC123
```

After that, install all the dependencies:

```ps
npm install
```

Then, start the bot:

```ps
npm run start
```

More scripts in the [package.json](https://github.com/Aqxorus/TS-bot-template/blob/main/package.json) file.
